# Локальная медиатека

Размещайте только проверенные безопасные файлы в каталогах `memes`, `gifs`, `photos`,
`reactions` и `welcome`. Для production предпочтительнее добавлять медиа через Telegram-команду
`/add_media`: бот сохранит устойчивый `file_id`, поэтому файл не придётся хранить в контейнере.

Не добавляйте порнографию, обнажённые изображения, сексуальный контент, шокирующее насилие,
незаконные материалы или сексуализированные изображения несовершеннолетних.
